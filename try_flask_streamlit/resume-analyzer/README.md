# Resume Analyzer Pro

**Resume Analyzer Pro** is a full-stack Python application to analyze resumes against job descriptions.  
It provides:

- Semantic match score between resume and job description
- Matched and missing skills detection
- Custom skills addition
- GPT-powered improvement suggestions
- Pie charts for skill match visualization
- Downloadable PDF report
- CSV logging of analysis results

Built with **Flask**, **Streamlit**, **Sentence Transformers**, **OpenAI GPT**, **Matplotlib**, and **ReportLab**.
